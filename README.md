# COMP217 SEC.401 - Final Project Phase 2 - Group 4

**For Unreal Engine - Version 5.7**

***RUN INSTRUCTIONS:***
- Download a .zip of the contents in the repository
- (Click the green [code] drop-down button, then select [Download ZIP]
- Extract the contents of the .zip into a new folder
- Run Unreal Engine
- Select [My Projects]
- Select [Browse]
- In the *File Explorer*, locate the downloaded project folder
- Select the *.uproject* file
- Wait for Unreal Engine to create build for project
- Once the editor opens, if scripts/blueprints do not load or function properly, recompile and wait
- Once the recompile process finishes, the project should function as intended
